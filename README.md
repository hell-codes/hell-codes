## ✦ &nbsp;What I'm Building — prabinyadav.dev

<br/>

> My portfolio is the project I'm most proud of. Built entirely from scratch — zero frameworks, zero shortcuts — every single feature written in pure HTML, CSS, and Vanilla JavaScript.

<br/>

<table width="100%">
<tr>
<td width="50%" valign="top">

**🌊 7 Unique Canvas Animations**
Every page has its own hand-coded animated background using the HTML5 Canvas 2D API — particles, hexagon grids, sine waves, glowing orbs, falling lines, rising dots.

**⚡ Zero Dependencies**
No React. No Tailwind. No webpack. No npm packages. `open index.html` — it just works.

**🔐 Real Admin Dashboard**
Firebase Firestore + Authentication. Add, feature, or delete projects live from a protected admin panel. Changes appear worldwide instantly.

</td>
<td width="50%" valign="top">

**🖱️ Custom UX System**
Built from scratch in Vanilla JS — custom cursor with trailing ring, scroll progress bar, IntersectionObserver scroll reveals, animated skill bars.

**🖼️ Image Compression Pipeline**
Canvas → JPEG compression before Firestore upload. Solves the 1MB document limit without any external library.

**📱 Fully Responsive**
Fixed floating hamburger, dropdown nav, `clamp()` fluid typography, adaptive canvas — all without Bootstrap or Tailwind.

</td>
</tr>
</table>
